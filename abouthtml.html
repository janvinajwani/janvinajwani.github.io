Define HTML

  • HTML stands for Hypertext Markup Language
  • HTML describes the structure of a Web page
  • HTML can be assisted by technologies such as Cascading Style Sheets (CSS) and scripting languages such as JavaScript.
  • Web browsers receive HTML documents from a web server or from local storage and render the documents into multimedia web pages.
  • HTML consists of a series of elements
  • HTML elements tell the browser how to display the content

HTML Elements

It is a type of HTML component which add semantics and formatting to parts of document (e.g., make text bold, organize it into paragraphs, lists and tables, or embed hyperlinks and images). Each element can have HTML attributes specified. Elements can also have content, including other elements and text. Each element is shown by the tag . To start an element we use opening tag and to end an element we use closing tag .
Here is the list of various tags:

Tags Description
< a > Defines a hyperlink
< b > Bolds a text
< br > To break a line
< del > To show a deleted text
< div > Div tag defines a section or box in html file in order to style that section through CSS.
< em > Defines emphasized text
< Head > To keep meta data
< i > Defines italic text
< ins > To Underline a text
< mark > Highlights a text
< p > For writing paragraphs
< pre > Defines a preformatted text
< strong > Defines important text
< sub > Defines subscript
< sup > Defines superscript

HTML Attributes

Attributes provide additional information about an element. Attributes are always specified in the start tag and usually come in name/value pairs like: name="value".
List of various attributes

Attribute Description
alt Specifies an alternative text for an image, when the image cannot be displayed
href Hypertext REFerence is used to create a link to another page
src specify the URL of the media resource
style Specifies an inline CSS style for an element
title used to display a tooltip

HTML Styles

Styling of HTML elements can be done with the help of style attribute.
Syntax of style attribute is: < tagname style="property:value";>

List of properties:

Property Description
background-color To set background color of HTML elements
color Defines the text color for an HTML element
font-family Defines the font of text
font-size Sets the text size
text-align Sets the text alignment

HTML Comments

HTML comment begins with <-- and ends with -->. Comments are visible to anyone that views the page source code , but are not rendered when the HTML document is rendered by a browser.

HTML CSS

CSS stands for Cascading Style Sheets.It is used for styling various HTML elements.
CSS can be added to HTML elements in 3 ways:

Inline By using the style attribute in HTML elements
Internal By using style element in head section of html file
External By using an external CSS file and link it in head section of html file

HTML Classes

Class attribute is used to give same styling effects to all the HTML elements having same class name. To select elements with a specific class for styling , write a '.' character, followed by the name of class.

HTML ID

ID stands for identifiers . The id attribute is a unique identifier which is used to specify the element in the HTML document. In CSS, to select an element with a specific id, # symbol is used followed by the id of the element.
